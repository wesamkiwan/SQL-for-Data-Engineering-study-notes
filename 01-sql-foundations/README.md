# Part 1 — SQL Foundations

*Goal: by the end of this part, you can read and write correct, useful SQL
queries against a real, multi-table schema.*

Every module here builds on the last, and all of them query the shared
[NorthStar Retail dataset](../datasets/). Make sure you've completed
[Part 0 — Orientation](../00-orientation/) and can successfully run the query
at the end of it before starting here.

## Modules

1. [Databases 101](01-databases-101/) — what a database, RDBMS, table, row,
   column, key, and schema actually are
2. [Basic Queries](02-basic-queries/) — `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`
3. [Filtering & Operators](03-filtering-and-operators/) — `BETWEEN`, `IN`,
   `LIKE`, and correctly handling `NULL`
4. [Aggregations](04-aggregations/) — `GROUP BY`, `HAVING`, and the aggregate functions
5. [Joins](05-joins/) — combining tables correctly with `INNER`/`LEFT`/`RIGHT`/`FULL`/`SELF` joins
6. [Subqueries & CTEs](06-subqueries-and-ctes/) — breaking big problems into named steps
7. [Set Operations](07-set-operations/) — `UNION`, `INTERSECT`, `EXCEPT`
8. [String, Date & Numeric Functions](08-string-date-numeric-functions/)
9. [CASE & Conditional Logic](09-case-and-conditional-logic/)
